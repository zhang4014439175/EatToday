# EatToday H5 Preview

这个目录是浏览器预览壳，用来在没有微信开发者工具的情况下先验证产品流程、文案和交互。

它不会影响微信小程序发布：

* 微信小程序源码仍在 `frontend/`。
* 微信开发者工具导入 `frontend/` 时，不会读取根目录下的 `preview/`。
* `preview/` 是普通 H5 静态页面，不参与小程序构建。

## 使用方式

完整可交互 H5 预览页：

```text
preview/index.html
```

最简单方式：直接用浏览器打开：

```text
preview/index.html
```

如果浏览器限制本地文件，可以用任意静态服务器打开。例如：

```powershell
cd E:\PythonProject\study\EatToday
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173/preview/
```

## 可测试流程

* 今日页：查看配对状态、今日食物、今日去哪玩、最近纪念日。
* 吃什么：添加食物、转盘抽取、锁定今日食物、双人盲盒投票。
* 去哪玩：新增去哪玩提案、接受、拒绝、修改建议、愿望单转行程。
* 我的：输入 `TA520` 配对、解除配对、新增/删除纪念日。

## 注意

这个 H5 预览不是小程序运行时，不验证以下能力：

* `wx.login`
* 订阅消息
* 小程序 TabBar 行为
* 真机样式差异
* 上传和发布

正式发布前仍然需要用微信开发者工具或找一台能安装它的电脑做最终验证。
